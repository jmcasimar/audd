# Código de Conducta

## Nuestros Estándares

Estamos comprometidos a proporcionar un ambiente acogedor y profesional.

**Comportamiento esperado:**
- Sea respetuoso e inclusivo
- Acepte retroalimentación constructiva
- Enfóquese en lo que beneficia al proyecto
- Muestre empatía hacia los demás

**Comportamiento inaceptable:**
- Acoso o lenguaje discriminatorio
- Ataques personales
- Comentarios trolleantes o inflamatorios
- Publicación de información privada de otros

## Aplicación

Los mantenedores del proyecto atenderán las violaciones de manera rápida y justa. Las consecuencias pueden incluir advertencias, suspensiones temporales o remoción permanente del proyecto.

## Reportes

Reporte inquietudes a los mantenedores del proyecto mediante issues de GitHub o correo electrónico.
