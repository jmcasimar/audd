# Code of Conduct

## Our Standards

We are committed to providing a welcoming and professional environment.

**Expected behavior:**
- Be respectful and inclusive
- Accept constructive feedback
- Focus on what benefits the project
- Show empathy towards others

**Unacceptable behavior:**
- Harassment or discriminatory language
- Personal attacks
- Trolling or inflammatory comments
- Publishing others' private information

## Enforcement

Project maintainers will address violations promptly and fairly. Consequences may include warnings, temporary bans, or permanent removal from the project.

## Reporting

Report concerns to project maintainers via GitHub issues or email.
